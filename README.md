# react-textalive-lyricapp-sample

React + TextAliveを使ったリリックアプリケーションのサンプルです。

[SUPARHERO](https://github.com/tosuri13/suparhero)からテンプレートを持ってきたので、Reactと言いつつNext.jsで動いてます。リリックアプリを作成する時の参考になれば幸いです。

## 🤔 Usage

1. 開発者トークンの埋め込み

ルートディレクトリに`.env.local`ファイルを作成し、以下の内容を書き込んでください。

```
NEXT_PUBLIC_TEXTALIVE_TOKEN={開発者トークン}
```

2. 依存関係のインストール

```shell
$ npm install
```

3. 開発用サーバの立ち上げ

```shell
$ npm run dev
```

ブラウザから`http://localhost:3000`にアクセスすることで、リリックアプリケーションの画面が表示されます。
